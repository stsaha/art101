I also did some experimenting with how the built-in sort() function works. If given an input that contains both uppercase and lowercase letters, the function will give priority to the uppercase laters, sorting them first before considering the lowercase letters. As a result, an alphabetical sort may not always be achieved. A solution to this problem, however, is making all the letters lowercase or uppercase before it is further manipulated. In my case, I chose to use the built-in toLowerCase() function to make sure the username is lowercase in its entirety before it is sorted.
